# SilForeFans
Silver Forestの歌詞ポータルです。あまり資料は現時点では少ないですが、今後増やしていく予定です。
このプロジェクトの作成のきっかけは、[歌詞一覧](http://www.silver-forest.net/lyrics.htm)が公式サイトにあるにも関わらず、Geocities Music含むGeocitiesの閉鎖からか、すべてリンク切れで、まったく機能をはたしていないからです。
この状況には私も驚きました。ググっても、歌詞サイトごとにデザインが違い、とても悩みました。
そこでこのポータルサイトを作成して、悩みを解決しよう、というわけです。
